# Black-Hole-Coffee-Showcase
Showcase for my work on the ongoing development of the game _Black Hole Coffee_

_Black Hole Coffee_ is a 2D side-scrolling metroidvania made in Godot where the player controls an advanced coffee barista robot aboard a futuristic colony spaceship that's been overrun and is on a collision course with a black hole. The game is fundamentally designed around the fact that the player character is a machine with modular bodyparts, as well as the fact that the advanced AI controlling the player character is hard-coded to perceive all tasks through a lens of coffee creation and serving.

On this project, I have primarily worked as the technical producer, as well as a gameplay programmer.

_Black Hole Coffee_ is in active development, and is currently in the prototyping phase.
